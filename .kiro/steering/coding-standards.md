# Coding Standards

## JavaScript/React
- Use ES6+ features (arrow functions, destructuring, async/await)
- Use functional components with hooks (no class components)
- Keep components small and focused
- Use meaningful variable names

## API Design
- RESTful endpoints
- Consistent error handling
- JSON responses
- Proper HTTP status codes

## AI Integration
- Always use Gemini API via `@google/generative-ai`
- Handle API errors gracefully
- Provide fallback content when AI generation fails
- Keep prompts creative and context-aware

## Styling
- Embrace the nostalgic GeoCities aesthetic
- Use gradients, playful colors, and retro fonts
- Ensure responsive design
- Maintain readability despite creative styling
